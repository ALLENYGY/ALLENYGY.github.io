# 1.1.1. Eigenvalues and Eigenvectors

  • Definitionпјҡ
    Let рқҗҙ be an рқ‘ӣГ—рқ‘ӣ matrix.
    A scalar рқңҶ is said to be an eigenvalue of рқҗҙ if there exists a nonzero vector рқҗұвү 0 such that
рқҗҙрқҗұ=\lambdaрқҗұ

Such рқҗұвү 0 is said to be an eigenvector for рқҗҙ belonging to рқңҶ.

  • The pair (рқңҶ,рқҗұ) is sometimes called an eigenpair for рқҗҙ.
  • Eigenvalues and eigenvectors are only defined for square matrices
  • Eigenvectors must be ==nonzero==
  • Property
    • рқҗҙ is singular вҮ” рқңҶ=0 is an eigenvalue of рқҗҙ.
    • If рқҗҙ is nonsingular, then \lambda^{вҲ’1} is an eigenvalue of рқҗҙ^{вҲ’1}, and рқҗұ is an eigenvector of рқҗҙ^(вҲ’1) belonging to рқңҶ^(вҲ’1), i.e. рқҗҙ^(вҲ’1) рқҗұ=рқңҶ^(вҲ’1) рқҗұ.
    • For рқ‘ҡ\geq1, \lambda^рқ‘ҡ is an eigenvalue of рқҗҙ^рқ‘ҡ, and рқҗұ is an eigenvector of рқҗҙ^рқ‘ҡ belonging to \lambda^рқ‘ҡ, i.e. рқҗҙ^рқ‘ҡ рқҗұ=\lambda^рқ‘ҡ рқҗұ.
    • If рқҗҙ^2=рқҗҙ, what can рқңҶ be? Hint: рқҗҙ^2 рқҗұ=рқҗҙрқҗұ.
    • If рқҗҙ^30=O, what can рқңҶ be? Hint: рқҗҙ^30 рқҗұ=рқ‘Ӯрқҗұ

Find eigenvalue and eignvector

рқҗҙрқҗұ=\lambdaрқҗұвҹәрқҗҙрқҗұвҲ’\lambdaрқҗұ=0вҹә(рқҗҙвҲ’\lambdaрқҗј)рқҗұ=0вҮ”рқҗұвҲҲрқ‘Ғ(рқҗҙвҲ’\lambdaрқҗј)
  • рқ‘Ғ(рқҗҙвҲ’рқңҶрқҗј): eigenspace зү№еҫҒз©әй—ҙ corresponding to the eigenvalue рқңҶ.
    This has a nontrivial solution рқҗұ вү рқҹҺ if and only if we pick рқңҶ so that the square matrix (рқҗҙвҲ’рқңҶрқҗј) is singular.That is: $$detвҒЎ(рқҗҙвҲ’\lambdaрқҗј)=0.$$
    First use detвҒЎ(рқҗҙвҲ’рқңҶрқҗј)=0 to find all eigenvalues for рқҗҙ, and then use рқ‘Ғ(рқҗҙвҲ’рқңҶрқҗј) to find all eigenvectors for рқҗҙ.

  • Property:
    Let рқҗҙ be an рқ‘ӣГ—рқ‘ӣ matrix and рқңҶ be a scalar. The following statements are equivalent:

    • рқңҶ is an eigenvalue of рқҗҙ.
    • (рқҗҙвҲ’рқңҶрқҗј)рқҗұ=рқҹҺ has nontrivial solutions.
    • рқ‘Ғ(рқҗҙвҲ’рқңҶрқҗј)вү {рқҹҺ}
    • рқҗҙвҲ’рқңҶрқҗј is singular.
    • detвҒЎ(рқҗҙвҲ’рқңҶрқҗј)=0

  • рқ‘қ(рқңҶ)=detвҒЎ(рқҗҙвҲ’рқңҶрқҗј) is a degree рқ‘ӣ polynomial in variable рқңҶ, called the characteristic polynomial зү№еҫҒеӨҡйЎ№ејҸ.

  • detвҒЎ(рқҗҙвҲ’рқңҶрқҗј)=0: characteristic equation for рқҗҙ

# 1.2. в…ў Diagonalization

# 1.2.1. Theorem 4.3.1

If \lambda_1 , \lambda_2пјҢвҖҰ \lambda_рқ‘ҳ are distinct eigenvalues of an рқ‘ӣГ—рқ‘ӣ matrix рқҗҙ with corresponding eigenvectors рқҗұ_1,рқҗұ_2,вҖҰ,рқҗұ_рқ‘ҳ, then the vectors рқҗұ_1,рқҗұ_2,вҖҰ,рқҗұ_рқ‘ҳ are linearly independent.

# 1.2.2. Diagonalization

An рқ‘ӣГ—рқ‘ӣ matrix рқҗҙ is said to be diagonalizable if there exists a nonsingular matrix рқ‘Ӣ and a diagonal matrix рқҗ· such that

рқ‘Ӣ^{вҲ’1} рқҗҙрқ‘Ӣ=рқҗ·

We say that рқ‘Ӣ diagonalizes рқҗҙ.

# 1.2.3. Theorem 4.3.2

An рқ‘ӣГ—рқ‘ӣ matrix рқҗҙ is diagonalizable if and only if A has рқ‘ӣ linearly independent eigenvectors.

  • If рқҗҙ is diagonalizable, then the column vectors of the diagonalizing matrix рқ‘Ӣ are eigenvectors of рқҗҙ and the diagonal elements of рқҗ· are the corresponding eigenvalues of рқҗҙ.
  • The diagonalizing matrix рқ‘Ӣ is not unique. Reordering the columns of a given diagonalizing matrix рқ‘Ӣ or multiplying them by nonzero scalars will produce a new diagonalizing matrix.
  • If рқҗҙ is рқ‘ӣГ—рқ‘ӣ and рқҗҙ has рқ‘ӣ distinct eigenvalues, then рқҗҙ is diagonalizable. If the eigenvalues are not distinct, then рқҗҙ may or may not be diagonalizable, depending on whether рқҗҙ has рқ‘ӣ linearly independent eigenvectors.
  • If рқҗҙ is diagonalizable, then рқҗҙ can be factored as a product

# 1.2.4. Symmertric Matrix

Let рқҗҙ be a symmetric рқ‘ӣГ—рқ‘ӣ matrix: рқҗҙ^рқ‘Ү=рқҗҙ.

  • Property:
    • (рқҗұ)^рқ‘Ү (рқҗҙрқҗІ)=(рқҗҙрқҗұ)^рқ‘Ү (рқҗІ).
    • If рқҗұ is an eigenvector of рқҗҙ and рқҗІвҠҘрқҗұ, then рқҗҙрқҗІвҠҘрқҗұ. [1]
    • If рқҗұ_1,рқҗұ_2 are eigenvectors of рқҗҙ with \lambda_1вү \lambda_2, then рқҗұ_1вҠҘрқҗұ_2. [2]

# 1.2.5. Orthogonally Diagonalizable

A square matrix рқҗҙ is ==orthogonally== diagonalizable if рқ‘„^рқ‘Ү рқҗҙрқ‘„=рқҗ· for some diagonal matrix рқҗ· and some orthogonal matrix рқ‘„.

  • рқ‘ӣГ—рқ‘ӣ matrix рқҗҙ is orthogonally diagonalizable if and only if рқҗҙ has рқ‘ӣ orthonormal eigenvectors (columns of рқ‘„)
  • From the previous slide, if ==a symmetric matrix рқҗҙ has рқ‘ӣ distinct eigenvalues==, then рқҗҙ has ==рқ‘ӣ mutually orthogonal eigenvectors==.
  • Surprisingly, symmetric рқ‘ӣГ—рқ‘ӣ matrix рқҗҙ ALWAYS has рқ’Ҹ orthonormal eigenvectors, and ALL eigenvalues are real!

# 1.2.6. Spectral Theorem for Real Symmetric Matrices 6.3.3

If рқҗҙ is a real symmetric matrix, then there is an orthogonal matrix рқ‘„ that diagonalizes рқҗҙ; that is, рқ‘„^рқ‘Ү рқҗҙрқ‘„=рқҗ·, where рқҗ· is diagonal, with real entries.

# 1.3.2. The Singular Value Decomposition

Assume that A is an mГ—n matrix with m вүҘ n. UОЈV^T is SVD of A
Factorize A into a product UОЈV^T , where

  • U is an m Г— m orthogonal matrix вҶ’UTU=Im\rightarrow U^TU=I_mвҶ’UвҖӢTвҖӢвҖӢU=IвҖӢmвҖӢвҖӢ
  • V is an n Г—n orthogonal matrix вҶ’VTV=In\rightarrow V^TV=I_nвҶ’VвҖӢTвҖӢвҖӢV=IвҖӢnвҖӢвҖӢ
  • ОЈ is an m Г—n matrix whose off-diagonal entries are all 0вҖҷs and whose diagonal elements satisfy: $$\sigma_1>=\sigma_2>=...=\sigma_n>=0$$
A = UОЈV^T

Properties

  • The first r columns of V is an orthonormal basis for R(AT)R(A^T)R(AвҖӢTвҖӢвҖӢ)
  • The rest of columns of V is an orthonormal basis for N(A)N(A)N(A)
  • The first r columns of U is an orthonormal basis for R(A)R(A)R(A)
  • The rest of columns of U is an orthonormal basis for N(AT)N(A^T)N(AвҖӢTвҖӢвҖӢ)

# 1.3.3. Theorem 4.5.1

If A is an m Г— n matrix, then A has a singular value decomposition.
