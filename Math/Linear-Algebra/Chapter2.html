# Linear Algebra

# 1. Chapter 2 Linear Transformations

# 1.1. в…  Linear Transformation

# 1.1.1. Linear Transformation

  • Definition:
    A mapping рқҗҝ from a vector space рқ‘ү into a vector space рқ‘Ҡ is said to be a linear transformation if
рқҗҝ(\alphaрқҗҜ_1+ \betaрқҗҜ_2 )= \alphaрқҗҝ(рқҗҜ_1 )+ \betaрқҗҝ(рқҗҜ_2 )

for all рқҗҜ_1,рқҗҜ_2вҲҲ рқ‘ү and for all scalars рқӣј and рқӣҪ.

  • Alternative definition:
    рқҗҝ is a linear transformation if and only if for all рқҗҜ_1,рқҗҜ_2вҲҲ рқ‘ү and scalars рқӣј,
рқҗҝ(рқҗҜ_1+рқҗҜ_2) = рқҗҝ(рқҗҜ_1) + рқҗҝ(рқҗҜ_2) рқҗҝ(\alphaрқҗҜ) = \alphaрқҗҝ(рқҗҜ)

# 1.1.2. Maping

  • Definition:
    A mapping рқҗҝ from a vector space рқ‘ү into a vector space рқ‘Ҡ will be denoted
рқҗҝ : рқ‘ү вҶ’ рқ‘Ҡ

When the arrow notation is used, it will be assumed that рқ‘ү and рқ‘ү represent vector spaces.

#

  • Definition:
    When the vector spaces рқ‘ү and рқ‘Ҡ are the same, we will refer to a linear transformation рқҗҝ : рқ‘ү вҶ’ рқ‘ү as a linear operator on рқ‘ү. Thus, a linear operator is a linear transformation that maps a vector space рқ‘ү into itself.

рқҗҝрқ‘–рқ‘ӣрқ‘’рқ‘Һрқ‘ҹ рқ‘ңрқ‘қрқ‘’рқ‘ҹрқ‘Һрқ‘Ўрқ‘ңрқ‘ҹ вҠӮ рқ‘ҷрқ‘–рқ‘ӣрқ‘’рқ‘Һрқ‘ҹ рқ‘Ўрқ‘ҹрқ‘Һрқ‘ӣрқ‘ рқ‘“рқ‘ңрқ‘ҹрқ‘ҡрқ‘Һрқ‘Ўрқ‘–рқ‘ңрқ‘ӣ

рқҗҝ:рқ‘үвҶ’рқ‘ү \subset рқҗҝ:рқ‘үвҶ’рқ‘Ҡ

# 1.1.3. Linear Transformations from в„қ^рқ‘ӣ to в„қ^рқ‘ҡ

  • Definition:
    If рқҗҙ is any рқ‘ҡГ—рқ‘ӣ matrix, and define
рқҗҝ:в„қ^рқ‘ӣвҶ’в„қ^рқ‘ҡ

via

рқҗҝ(рқҗұ)=рқҗҙрқҗұ

for all рқҗұвҲҲв„қ^рқ‘ӣ. Then рқҗҝ is a linear transformation from в„қ^рқ‘ӣ to в„қ^рқ‘ҡ.

  • Property:
    If рқҗҝ is a linear transformation mapping a vector space рқ‘ү into a vector space рқ‘Ҡ, then
    • рқҗҝ(рқҹҺ_рқ‘ү) =рқҹҺ_рқ‘Ҡ (where рқҹҺ_рқ‘ү and рқҹҺ_рқ‘Ҡ are the zero vectors in рқ‘ү and рқ‘Ҡ, respectively).
    • If рқҗҜ_1,вҖҰ,рқҗҜ_n are elements of рқ‘ү and \alpha_1,вҖҰ,\alpha_рқ‘ӣ are scalars, then рқҗҝ(\alpha_1 рқҗҜ_1+\alpha_2 рқҗҜ_2+вҖҰ+\alpha_рқ‘ӣ рқҗҜ_n)=\alpha_1 рқҗҝ(рқҗҜ_1 )+\alpha_2 рқҗҝ(рқҗҜ_2 )вҖҰ+\alpha_рқ‘ӣ рқҗҝ(рқҗҜ_n)
    • рқҗҝ(вҲ’рқҗҜ)= вҲ’рқҗҝ(рқҗҜ) for all рқҗҜвҲҲ рқ‘ү.

# 1.1.4. Kernel

  • Definition:
    Let рқҗҝ : рқ‘ү вҶ’ рқ‘Ҡ be a linear transformation. The kernel of рқҗҝ, denoted kerвҒЎ(рқҗҝ), is defined by
    • kerвҒЎ(рқҗҝ) = {рқҗҜвҲҲ рқ‘ү | рқҗҝ(рқҗҜ) = рқҹҺ_рқ‘Ҡ}
  • The kernel of рқҗҝ is the set of all vectors from рқ‘ү that рқҗҝ transforms to the zero vector in рқ‘Ҡ.
  • Note, if рқҗҝ is defined via matrix multiplication for some matrix рқҗҙ, then kerвҒЎ(рқҗҝ)=рқ‘Ғ(рқҗҙ), the null space of рқҗҙ.

# 1.1.5. Image

  • Definition:
    Let рқҗҝ : рқ‘ү вҶ’ рқ‘Ҡ be a linear transformation and let рқ‘Ҷ be a subspace of рқ‘ү. The image of рқ‘Ҷ, denoted рқҗҝ(рқ‘Ҷ), is defined by
    рқҗҝ(рқ‘Ҷ) = {рқҗ°вҲҲ рқ‘Ҡ |рқҗ°= рқҗҝ(рқҗҜ) for some $ рқҗҜвҲҲ рқ‘Ҷ$}

# 1.1.6. Range

  • Definition:
    The image of the entire vector space, рқҗҝ(рқ‘ү), is called the range of рқҗҝ.

In other word

  • If {рқҗҜ_1, рқҗҜ_2,вҖҰ,рқҗҜ_рқ‘ҳ} is a spanning set of рқ‘Ҷ, then рқҗҝ(рқ‘Ҷ) is the span of рқҗҝ(рқҗҜ_1), рқҗҝ(рқҗҜ_2),вҖҰ рқҗҝ(рқҗҜ_рқ‘ҳ).
  • If рқҗҝ is defined via matrix multiplication for some matrix рқҗҙ, then рқҗҝ(рқ‘ү), the range of рқҗҝ, is the column space of рқҗҙ.

# 1.1.7. Theorem 2.1.1 (Kernel/Image are subspaces)

If рқҗҝ : рқ‘ү вҶ’ рқ‘Ҡ is a linear transformation and рқ‘Ҷ is a subspace of рқ‘ү. Then

  • kerвҒЎ(рқҗҝ)={рқҗҜвҲҲрқ‘ү:рқҗҝ(рқҗҜ)=рқҹҺ_рқ‘Ҡ} is a subspace of рқ‘ү.
  • рқҗҝ(рқ‘Ҷ)={рқҗ°вҲҲрқ‘Ҡ:рқҗ°=рқҗҝ(рқҗҜ) for some рқҗҜвҲҲрқ‘Ҷ} is a subspace of рқ‘Ҡ.

# 1.2. в…Ў Matrix Representations of Linear Transformations

# 1.2.1. Theorem 2.2.1 (Standard matrix representation)

If рқҗҝ is a linear transformation mapping в„қ^рқ‘ӣ into в„қ^рқ‘ҡ, then there is an рқ‘ҡГ—рқ‘ӣ matrix рқҗҙ, such that for each xвҲҲв„қ^рқ‘ӣ, рқҗҝ(рқҗұ)=рқҗҙрқҗұ.
In fact, the рқ‘—^{рқ‘Ўв„Һ} column vector of рқҗҙ is given by $$рқ’Ӯ_рқ‘—=рқҗҝ(рқ’Ҷ_рқ‘—)$$(рқ‘—=1,2,вҖҰ,рқ‘ӣ)

  • How to write the рқ‘ҡГ—рқ‘ӣ matrix рқҗҙ:
    • See what рқҗҝ does to the basis vectors рқҗҜ_1, рқҗҜ_2,вҖҰ,рқҗҜ_рқ‘ӣ in рқ‘ү. Because рқҗҝ(рқҗҜ_1) is in рқ‘Ҡ, write it as linear combination of рқҗ°_1, рқҗ°_2,вҖҰ,рқҗ°_рқ‘ҡ;
    • write the coordinate vector [рқҗҝ(рқҗҜ_1)]_рқҗ№ as the first column of matrix рқҗҙ.
    • Repeat for рқҗҝ(рқҗҜ_2),рқҗҝ(рқҗҜ_3)вҖҰ, to find the 2nd,3rd2_{nd}, 3_{rd}2вҖӢndвҖӢвҖӢ,3вҖӢrdвҖӢвҖӢ,вҖҰ column of рқҗҙ.
