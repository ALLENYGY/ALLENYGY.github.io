# [[USTC-L1]]

## **互联网**

- 网络
	- 节点与边构成的大小形状无关的拓扑
	- 电话网，社交网络....
- 计算机网络---具体构成角度
	- 联网计算机构成的系统
	- 节点
		- 主机节点，及其运行的应用程序 ⬛
		- 数据交换节点 ⭕️
			- 路由器、交换机等网络设备 既不是源也不是目标，转发数据
			- 根据层次不同可分
				- 中继器
				- 交换机 链路层
				- 路由器 网络层
				- 负载均衡设备 更高层
	- 边：通信链路
		- 接入网链路^[Access]：主机连接到互联网的链路
		-  骨干链路^[Backbone]：路由器间的链路
	- 协议：
		- 对等层实体在通信过程当中的应该遵守的规则的集合，包括语法、语义和时序
		- 协议定义了两个或多个通信实体之间交换的报文格式和次序，以及在报文传输和/或接收/或其他事件方面所采取的动作
			- PDU：协议数据单元
		- 按照层次不同
			- 物理层协议
			- 链路层协议
				- 网络层协议
					- IP
				- 传输层协议
					- TCP
					- UDP
				- 应用层协议

> [!abstract]+ 互联网：从具体组成角度
> “网络的网络”
> - 数以亿计、互联的计算设备
> 	- 主机 = 端系统 end system host
> 	- 运行网络应用程序
>  - 通信链路
>  	- 光纤、同轴电缆、无线电、卫星
>  	- 传输速率=带宽（bps）bit / second
> - 分组交换设备：转发分组（packets）
> 	- 路由器和交换机

- Internet 标准
	- RFC: Request for comments
	- IEFC: Internet Engineering Task Force

> [!abstract]+ 互联网： 从服务角度
> 1. 使用通信设施进行通信的分布式应用
> 	- Web、VoIP、email、分布式游戏....
> 2. 通信基础设施为 apps 提供编程接口（通信服务）
> 	- 将发送和接收数据的 apps 与互联网连接起来
> 	- 为 app 应用提供服务选择类似与邮政服务
> 		- 无连接 不可靠的服务
> 		- 面相连接的可靠服务
