# Judgment questions

## 1.4

- 1. The set {$[1,0,0]^𝑇, [0,1,0]^𝑇$} is a basis for a two-dimensional subspace of $ℝ^3$.  ==True==
- 2. The set {$[1,0,0]^𝑇$} is a basis for a one-dimensional subspace of $ℝ^3$.  ==True==
- 3. The set of all vectors of the form $[𝑎,2𝑎,𝑏]^𝑇$ is a subspace spanned by the vectors $[1,2,0]^𝑇$ and $[0,0,1]^𝑇$.  ==True==
- 4. The set of vectors of the form $[𝑎,𝑏,2𝑎−𝑏]^𝑇$ is a vector in the plane spanned by the vectors $[1,0,2]^𝑇$ and $[0,1,−1]^𝑇$.  ==True==
- 5. The set {$[1,0,0]^𝑇, [0,1,0]^𝑇, [1,1,0]^𝑇$} is a basis for a subspace of $ℝ^3$.  ==False==
  
## 1.6

- 1. If 𝑆 is a subspace of a vector space 𝑉, then 𝑆 is a vector space.  ==True==
- 2. It is possible to find a pair of two-dimensional subspaces 𝑆 and 𝑇 of $ℝ^3$ such that 𝑆∩𝑇={𝟎}.  ==False==
- 3. If 𝑆 and 𝑇 are different proper subspaces of a vector space 𝑉, then 𝑆∪𝑇 is a subspace of 𝑉.  ==False==  
- 4. If 𝑆 and 𝑇 are subspaces of a vector space 𝑉, then 𝑆∩𝑇 is a subspace of 𝑉.  ==True==
- 5. If $𝐱_1,𝐱_2,…,𝐱_𝑛$ span $ℝ^𝑛$, then they are linearly independent.  ==True==
- 6. If $𝐱_1,𝐱_2,…,𝐱_𝑛$ span a vector space 𝑉, then they are linearly independent.  ==False==
- 7. If $𝐱_1,𝐱_2,…,𝐱_𝑛$ are vectors in vector space 𝑉 and $Span(𝐱_1,𝐱_2,…,𝐱_𝑛)$=$Span(𝐱_1,𝐱_2,…,𝐱_(𝑛−1))$ then $𝐱_1,𝐱_2,…,𝐱_𝑛$ are linearly dependent.  ==True==
- 8. If 𝐴 is an 𝑚×𝑛 matrix, then 𝐴 and $𝐴^𝑇$ have the same rank.  ==True==
- 9. If 𝐴 is an 𝑚×𝑛 matrix, then 𝐴 and $𝐴^𝑇$ have the same nullity.  ==False==
